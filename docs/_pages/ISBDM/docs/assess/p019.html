Describing the manifestation

Describing the manifestation

The following workflow is recommended for the creation of a description of a single manifestation after preliminary assessment of a resource.

  1. How does the manifestation describe itself?
    1. ✽Record has manifestation statement elements
      1. ✽Record has manifestation statement of title and responsibility elements
    2. Record has note on manifestation statement elements
  2. What appellations are used to label the manifestation?
    1. ✽Record has appellation of manifestation elements
      1. Record has title of manifestation elements
      2. Record has identifier of manifestation elements
      3. Record has access point of manifestation elements
        1. ✽Record has authorized access point of manifestation elements
      4. Record has note on nomen associated with manifestation elements
  3. What are the details of the creation of the manifestation?
    1. Record has creator agent of manifestation elements
    2. Record has note on agent associated with manifestation elements
    3. Record has place of creation of manifestation elements
    4. Record has note on place associated with manifestation elements
    5. Record has date of creation of manifestation elements
    6. Record has note on time-span associated with manifestation elements
  4. What are the details of the carrier, including extent?
    1. ✽Record a has unitary structure element
    2. Record has sub-unit elements
    3. Record has super-unit elements
    4. ✽Record has category of carrier elements
    5. ✽Record has media type elements
    6. Record has physical characteristic elements
    7. Record has note on category of manifestation elements
    8. Record has extent of manifestation elements
    9. Record has item that exemplifies manifestation elements
  5. What are the details of the content that is embodied in the manifestation?
    1. Record has expression embodied in manifestation elements
    2. Record has work embodied in manifestation elements
    3. ✽Record has category of embodied content elements
    4. Record has note on expression associated with manifestation elements
    5. Record has note on work associated with manifestation elements
  6. How is the manifestation related to other resource entities of the same resource?
    1. ✽Record has expression embodied in manifestation elements
    2. ✽Record has work embodied in manifestation elements
  7. How is the manifestation related to other entities?
    1. Record relationship elements
      1. ✽Record has agent associated with manifestation elements
        1. ✽Record has creator agent of manifestation elements
          1. ✽Record has creator collective agent of manifestation elements
          2. ✽Record has creator person of manifestation elements
      2. ✽Record has place associated with manifestation elements
        1. ✽Record has place of creation of manifestation elements
      3. ✽Record has time-span associated with manifestation elements
        1. ✽Record has date of creation of manifestation elements
    2. Record other has entity associated with manifestation elements
    3. Record has note on entity associated with manifestation elements
    4. Record a is reference source of element
    5. Record a is subject manifestation of element
  8. What other information is considered to be useful for users of the metadata?
    1. Record attribute elements
    2. Record has note on manifestation elements
