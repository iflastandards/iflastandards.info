Display of ISBDM metadata

ISBD for Manifestation does not prescribe a selection, order, layout, or punctuation for displaying the values of elements in a description of a manifestation.

The choice of methods for displaying ISBDM metadata in an information retrieval application is influenced by the intended Metadata utility and processing of ISBDM elements.

The value of a has manifestation statement element or element sub-type is at a broad level of granularity, and it may overlap with the value of another similar element.

Specific element labels may be omitted to simplify the display of a set of values.

Examples

The values of statements in a description are concatenated into a single display value with delimiting punctuation:

"Jane Austen, Emma, edited with an introduction by Ronald Blythe. - Penguin English library. - Penguin Books Ltd, Harmondsworth … England, … Reprinted 1971. - ISBN 0 14 043.101 5."

[Full example: Emma (1971; Penguin Books; volume). The delimiting punctuation is ". - " (stop-space-hyphen-space).]

"give it up b/w tired. - lcd soundsystem | a. written by james murphy b. by james murphy and pat mahoney | a give it up | james murphy: vocals, bass, drums, percussion, synthesizers, programming | produce by the dfa | b tired | james murphy: vocals, bass, guitar; pat mahoney: drums, percussion; ems synthesizer | produce by the dfa. - p+c dfa records 2002. - output recordings … london, - oprdfa004. - 45 rpm"

[Full example: Give it up b/w Tired (2002; Ouput Recordings; audio disc). The delimiting punctuation is ". - " (stop-space-hyphen-space).]

"Henri Cartier-Bresson (1908-2004) / Truman Capote, ecrivain, USA, 1947 / courtesy Fondation HCB : © Nouvelles Images S.A. et H. Cartier-Bresson - Magnum Photos 2008 / 45700 Lombreuil, France : 764697001022, 764697150065"

[Full example: Truman Capote, ecrivain, USA, 1947 (2008; Nouvelles Images S.A.; card). The delimiting punctuation is " :" (space-colon-space).]

The controlled value of an attribute element can be displayed or otherwise output as a non-textual symbol.

Examples

Display the value "volume" of a has category of manifestation element as .

Display the value "microform" of a has media type element as .

The controlled values of a has category of manifestation element and element sub-types can be concatenated into a single compound value and displayed with or without a generic "category" label to simplify the display of a set of values.

Examples

"volume; unmediated; still image; text"

[Full example: Asterix and the gowden heuk (2014; Itchy Coo; volume). The delimiting punctuation is "; " (semicolon-space).]

"Type: audiocassette - spoken word"

[Full example: Harry Potter and the goblet of fire (2000; Listening Library; audiocassette). The delimiting punctuation is " - " (space-hyphen-space). The value of has media type is omitted for clarity. "Type" is used as a generic display label.]

The controlled values of a has creator agent of manifestation, has place of creation of manifestation, and has date of creation of manifestation elements and element sub-types can be concatenated into a single compound value and displayed with or without a generic "publication" label to simplify the display of a set of values.

Examples

"London (England) : Methuen Children’s Books, 1986"

[Full example: House at Pooh Corner (1986; Methuen Children’s Books; volume). The order of elements and delimiting punctuation conform to ISBD Area 4.]

"Published: Cham (Switzerland); Springer (Cham, Switzerland); 2022-"

[Full example: Springer handbooks of political science and international relations (2022-; Springer; volume). The order of elements conforms to ISBD Area 4. The delimiting punctuation is "; " (semicolon-space). "Published" is used as a generic display label.]
