Manifestation SES

This is a suggested string encoding scheme for an authorized access point of the manifestation and of a manifestation that is related to the manifestation.

The encoding scheme is applicable to the value of a has access point of manifestation element or element sub-type or a has manifestation associated with manifestation element or element subtype.

The base access point of a manifestation is a normalized value of has title proper. If there is no such value, a title supplied by the cataloguing agency may be used.

The base access point is followed by a set of qualifiers enclosed in brackets "(…)" and separated by semicolons "; ".

The ISBDM string template is:

normalised({has title proper})|[supplied title] + " (" + {has date of creation of manifestation}|{has date of copyright of manifestation} + "; " + {has creator agent of manifestation} + "; " + {has category of carrier} + ")"

If a value of a qualifier is not known, omit the qualifier from the access point.

If a qualifier has more than one value, use a concatenation of the values separated by " + " (space-plus-space). Order the values by prominence and quantity.

The sequence of qualifiers supports a standard secondary collocation and sort order of authorized access points in a browse index for manifestations.

Examples

has authorized access point of manifestation
"Desserte rouge 1908 (1993; Editions du Centre Pompidou; sheet)"
[Full example: Desserte rouge 1908 (1993; Editions du Centre Pompidou; sheet).]
has authorized access point of manifestation
"Corso delli fiumi Drino, e Boiana nella Dalmatia (1688; fletë)"
[Full example: Corso delli fiumi Drino, e Boiana nella Dalmatia (1688; fletë). The publisher is not known.]
has authorized access point of manifestation
"Съкровищница (2018; Народна библиотека “Св. св. Кирил и Методий”; том + компютърен диск)"
[Full example: Съкровищница (2018; Народна библиотека “Св. св. Кирил и Методий”; том + компютърен диск). The manifestation has sub-units that have different categories of carrier.]

Add qualifiers to the sequence to disambiguate the basic qualified access point.

Added qualifiers are conditional on specific characteristics of the manifestation that are recorded as attribute elements. They support a tertiary collocation and sort order of authorized access points.

Qualify a value of has category of carrier that is "volume" with a value of has binding.

Examples

has authorized access point of manifestation
"Paris apartment (2022; HarperCollinsPublishers; volume; case binding)"
[Full example: Paris apartment (2022; HarperCollinsPublishers; volume; case binding).]
has authorized access point of manifestation
"Evangelistarium Marci Maruli Spalatensis (201-; Nacionalna i sveučilišna knjižnica u Zagrebu; online jedinica građe)"
[Full example: Evangelistarium Marci Maruli Spalatensis (201-; Nacionalna i sveučilišna knjižnica u Zagrebu; online jedinica građe).]

Qualify a value of has category of carrier that is "online resource" with a value of has encoding format.

Examples

has authorized access point of manifestation
"Emma (1994; Project Gutenberg; online resource; plain text format)"
has alternate manifestation
"Emma (1994; Project Gutenberg; online resource; HTML format)"
has alternate manifestation
"Emma (1994; Project Gutenberg; online resource; EPUB3 format)"
has alternate manifestation
"Emma (1994; Project Gutenberg; online resource; EPUB format)"
has alternate manifestation
"Emma (1994; Project Gutenberg; online resource; Kindle format)"
[Full example: Emma (1994; Project Gutenberg; online resource; plain text format).]

Qualify a value of has category of carrier that is "videodisc" with a controlled term for a finer level of granularity of the category of carrier.

Examples

has authorized access point of manifestation
"Einstein on the beach (2016; Opus Arte; videodisc; DVD)"
has alternate manifestation
"Einstein on the beach (2016; Opus Arte; videodisc; BD)"
[Full example: Einstein on the beach (2016; Opus Arte; videodisc; DVD).]

Qualify a value of has title proper that is a generic title with an authorized access point of an agent who is responsible for the creation of works and expressions that are embodied in the manifestation.

Examples

has authorized access point of manifestation
"Summary records (Unesco. Executive Board. 215th session; 2022; Paris) (2023; Unesco Executive Board; svezak)"

Apply the same string encoding scheme to an authorized access point of a related manifestation.
