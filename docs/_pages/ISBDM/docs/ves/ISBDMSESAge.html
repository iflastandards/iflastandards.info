Agent SES

This is a suggested string encoding scheme for an authorized access point of an agent who is related to the manifestation.

The encoding scheme is applicable to the value of a has agent associated with manifestation element or element subtype.

For an agent who is a collective agent, apply the Collective Agent SES.

For an agent who is a person, apply the Person SES.

For any agent, add a qualifier that indicates that the agent is a collective agent or a person.
