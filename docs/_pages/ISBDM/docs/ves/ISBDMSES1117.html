Numeric designation SES

This is a suggested string encoding scheme for a value of a has numeric designation element.

For a manifestation that embodies the expression of a diachronic work, a value for the first numeric designation in the sequence is followed by a hyphen "-" to indicate an open sequence.

The ISBDM string template is:

first in sequence({has numeric designation}) + "-"

Examples

has numeric designation
"Heft 1-"
[Full example: Journal für Facility Management (2009-; Technische Universität Wien‏; online resource).]
has numeric designation
"Lecture #1-"
[Full example: Synthesis lectures on data, semantics, and knowledge (2021-; Morgan & Claypool Publishers; volume).]

For a manifestation that embodies the expression of a diachronic work, and if there is a change in the sequence of numeric designations of issues or iterations, values for the first and last numeric designations in the old sequence are separated by a hyphen "-" to indicate sequence range, followed by a value for new open sequence. Sequences are separated with semi-colons "; ".

The ISBDM string template is:

first in old sequence({has numeric designation}) + "-" + last in old sequence({has numeric designation}) + "; " + first in new sequence({has numeric designation}) + "-"

For a manifestation that embodies the expression of a diachronic work that is known or assumed to have terminated, values for the first and last numeric designations in the sequence are separated by a hyphen "-" to indicate sequence range.

The ISBDM string template is:

first in sequence({has numeric designation}) + "-" + last in sequence({has numeric designation})
