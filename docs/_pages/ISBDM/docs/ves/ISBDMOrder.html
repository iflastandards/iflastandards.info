Order of elements

This is a suggested order of presentation of element values for a "standard record" display:

  1. Representation of the manifestation. Transcribed values with relevant notes:
    1. has manifestation statement elements and element sub-types
    2. has note on manifestation statement elements and element sub-types
  2. Appellations of the manifestation. Controlled values and relevant notes:
    1. has nomen associated with manifestation elements and element sub-types
    2. has note on nomen associated with manifestation element
  3. Creation of the manifestation. Controlled values:
    1. has creator agent of manifestation elements and element sub-types
    2. has place of creation of manifestation elements and element sub-types
    3. has date of creation of manifestation elements and element sub-types
  4. Attributes of the manifestation. Controlled values with relevant notes:
    1. has category of manifestation element and element sub-types
    2. has note on category of manifestation element
    3. has physical characteristic element and element sub-types
    4. has note on physical characteristic element
    5. has use rights element and element sub-types
  5. Relationships with other manifestations. Controlled values with relevant notes:
    1. has manifestation associated with manifestation element and element sub-types
    2. has note on manifestation associated with manifestation element
  6. Relationships with other resource entities. Controlled values with relevant notes:
    1. has expression associated with manifestation element and element sub-types
    2. has note on expression associated with manifestation element
    3. has work associated with manifestation element and element sub-types
    4. has note on work associated with manifestation element
    5. has item associated with manifestation element and element sub-types
    6. has note on item associated with manifestation element
  7. Relationships with other agents. Controlled values with relevant notes:
    1. has distributor agent element and element sub-types
    2. has collective agent associated with manifestation element
    3. has person associated with manifestation element
    4. has agent associated with manifestation element
    5. has note on agent associated with manifestation element
  8. Relationships with other places and time-spans. Controlled values with relevant notes:
    1. has place associated with manifestation element and element sub-types
    2. has note on place associated with manifestation element
    3. has time-span associated with manifestation element and element sub-types
    4. has note on time-span associated with manifestation element
