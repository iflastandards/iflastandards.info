Order of elements

Extent of manifestation SES

This is a suggested string encoding scheme for a value of a has extent of manifestation element.

The string encoding scheme combines values of the element sub-types in the following order:

  1. has extent of unitary structure
  2. has extent of unit
  3. has extent of aggregated content
  4. has extent of embodied content

The ISBDM string template is:

{has extent of unitary structure} + "(" {has extent of unit} + "; " + {has extent of aggregated content} + " in " + {has extent of embodied content} + ")"

Examples

has extent of manifestation
"3 volumes (124 leaves; 150 photographs in 200 pages)"
has extent of manifestation
"3 volumes (150 photographs in 200 pages)"
has extent of manifestation
"3 volumes (200 pages)"

For a manifestation that has no sub-units, use the string that is the number "1" followed by a value for the has category of carrier element as a substitute for a value of has extent of unitary structure.

Examples

has extent of manifestation
"1 list (70 x 50 cm; 1 plakat na 70 x 50 cm)"
[Full example: Zadar does Banksy (2023; Narodni muzej Zadar; list).]

For a manifestation that is a super-unit, a string encoding scheme may combine values of element sub-types for sub-units of the manifestation.

Examples

has extent of manifestation
"1 volume (122 pages) + 1 audio disc (15 performed songs in 49 min 14 sec)"
[Full example: Mojo 357 (2023; H Bauer Publishing Ltd.; volume + audio disc).]

For a manifestation that embodies only one expression, use the string that is the number "1" followed by a value for the has category of embodied content element or a controlled term for a finer level of granularity of the category as a substitute for a value of has extent of aggregated content.

Examples

has extent of manifestation
"3 volumes (1 text in 363 pages)"
[Full example: Emma (1815; John Murray; case binding). The manifestation is a single novel that is embodied in three volumes.]
has extent of manifestation
"1 hartë"
[Full example: Corso delli fiumi Drino, e Boiana nella Dalmatia (2011; Biblioteka Kombëtare e Shqipërisë; burim online).]
