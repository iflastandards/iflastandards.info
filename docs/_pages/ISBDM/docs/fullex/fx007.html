Absurd drama (1978; Penguin Books; volume; perfect binding)

This example describes a printed volume of four plays, three of them translated into English, with an authored Introduction, 16 p. Translators’ names are provided in the Contents, a source of information for the manifestation itself, although not relevant as the information is recorded in a note. First published in Penguin Books in 1962 and reprinted several times, this manifestation is reprinted in 1978. The manifestation is a sub-unit of a series with no information about its termination.

The focus of the description is the manifestation and the super-unit publisher series is not described; it is a moot point to include a relationship to the series. A string encoding scheme for the has authorized access point of manifestation elemeent is specific for issues of a publisher series. The relationships are mediated in this example via authorized access points.

The manifestation is published in a single unit.

The manifestation is a collected expressions aggregate and an augmented expression aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
