Einstein on the beach (2016; Opus Arte; videodisc; DVD)

This example describes a set of videodiscs that embody a work that is a recording of a live opera. There is an accompanying booklet that is hardbound with disc trays inside the front and back cover. The text in the booklet is a set of brief essays, in parallel English, French, and German. The same content is also published on Blu-ray discs.

The manifestation is published in multiple sub-units.

The manifestation is an augmented expression aggregate and parallel expressions aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
