Harry Potter and the goblet of fire (2000; Listening Library; audiocassette)

This example describes a set of audiocassettes that embody an audio recording of a work of fiction. The cassettes are contained in a box.

The manifestation is published in multiple sub-units.

The manifestation is not an aggregate.

The manifestation embodies a static work.

The language of cataloguing is English.

The description is intended for a relational database application.
