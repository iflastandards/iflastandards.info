has manifestation statement of publication, production, manufacture, or distribution

Element reference

Definition
Relates a manifestation to a manifestation statement that includes dates and places of publication, production, manufacture, or distribution of the manifestation, and the names and functions of agents who are responsible for publication, production, manufacture, or distribution of the manifestation.
Scope note
This element includes roles of agents who are involved in any aspect of the creation and distribution of a manifestation. Creation roles include printing and other forms of mechanical reproduction. Distribution roles include bookselling and other forms of product retail. This element includes statements of copyright, license, or privilege.

Stipulations

Use the element super-type to record a less specific statement.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a single occurrence of the element that is selected from one or more values that are associated with the manifestation. Use the following order of preference:

  1. A statement about the publication of the manifestation.
  2. A statement about the production of the manifestation.
  3. A statement about the manufacture of the manifestation.
  4. A statement about the distribution of the manifestation.

Record additional occurrences of the element, if it is considered to be useful for users of the metadata.

Include information about only one of the aspects of publication, production, manufacture, and distribution if more than one aspect is covered by the statement in the manifestation. Use the same order of preference.

Include a name, place, and time-span that is associated with an agent who is responsible for publication, production, manufacture, or distribution of the manifestation.

Agents

Include a name that is associated in a manifestation statement with an agent who is responsible for publication, production, manufacture, or distribution of the manifestation.

Places

Include a place that is associated in a manifestation statement with an agent who is responsible for publication, production, manufacture, or distribution of the manifestation.

Include a place that is selected from more than one place that is associated with a single instance of an agent in a manifestation statement using the following order of preference:

  1. A place that is made more prominent by typography.
  2. The first place that appears in the statement.
  3. Places that are considered to be useful for users of the metadata.
  4. All places.

Include an address of a publisher, producer, manufacturer, or distributor, if it is considered to be useful for users of the metadata.

Include a name of a larger place of publication, production, manufacture, or distribution such as a country, state, etc., if it is considered to be useful for users of the metadata.

Time-spans

Include a date that is associated in a manifestation statement with the publication, production, manufacture, or distribution of the manifestation.

Include all dates of publication, production, manufacture, or distribution in different calendars that appear in a manifestation.

Include the year of date of publication, production, manufacture, or distribution. Include finer levels of date of publication, production, manufacture, or distribution such as day and month, if it is considered to be useful for users of the metadata.

Include a statement of date of copyright when it is different from the date of publication, if it is considered to be useful for users of the metadata.

Include a statement of the latest date of copyright if more than one date of copyright appears in the manifestation.
