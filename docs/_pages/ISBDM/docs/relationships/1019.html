has creator agent of manifestation

Element reference

Definition
Relates a manifestation to an agent who is responsible for its manufacture, production, or publication.
Scope note
Domain
Manifestation
Range
Agent
Element sub-type
has manufacturer agent has producer agent has publisher agent
has creator collective agent of manifestation has creator person of manifestation
Element super-type
has agent associated with manifestation

Additional information

Consider a manifestation that has a value of category of carrier of "online resource" to be published.

See also: has manifestation statement of publication, production, manufacture, or distribution

Stipulations

✽

If one or more creator agents are associated with the manifestation, record at least one occurrence of the element. Use the following order of preference to select the value:

  1. Agent who is a publisher of the manifestation
  2. Agent who is a producer of the manifestation
  3. Agent who is a manufacturer of the manifestation
