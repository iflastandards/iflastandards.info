has date of manufacture

Element reference

Definition
Relates a manifestation to a time-span during which the manifestation is manufactured.
Scope note
This element includes manufacture by printing.
Domain
Manifestation
Range
Time-span
Element sub-type
Element super-type
has date of creation of manifestation

Stipulations

Use the element super-type to record a less specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.

Examples

has date of manufacture
"1519"
[Full example: Evangelistarium Marci Maruli Spalatensis (1519; Officina Adamae Petri; svezak). The date is a year when the manifestation is printed.]
