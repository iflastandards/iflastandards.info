Element reference

Definition
Relates a manifestation to an item that exemplifies a previous manifestation and whose content is carried by the new manifestation.
Scope note
This element applies to an item that has been modified.
Domain
Manifestation
Range
Item
Element sub-type
Element super-type
has item associated with manifestation

Additional information

For a manifestation that does not include modifications of an item that exemplifies the manifestation that is copied, use a has manifestation reproduced by manifestation element.

Use a has manifestation associated with manifestation element to record the manifestation that is exemplified by the item that is copied.

See also: Reproductions

Stipulations

Use the element super-type to record a less specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.
