has collective agent associated with manifestation

Element reference

Definition
Relates a manifestation to a collective agent.
Scope note
This element includes a corporate body, a jurisdiction, and a family.
Domain
Manifestation
Range
Collective Agent
Element sub-type
has creator collective agent of manifestation has distributor collective agent
Element super-type
has agent associated with manifestation

Additional information

A collective agent may have a form of name that is usually associated with a person. The name may be qualified in an authorized access point to indicate that it is assigned to a collective agent.

Examples

has collective agent associated with manifestation
"Jethro Tull (Musical group)"

Element values

The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.

String encoding schemes

A string encoding scheme should construct the base access point from a name of the collective agent that is the most common in external sources of information.

A string encoding scheme should use the following strings as qualifiers of a base access point:

  • A controlled term for a category of the collective agent
  • A field of activity of the collective agent
  • A period of activity of the collective agent, including the date of a conference, meeting, or voyage
  • A place of activity of the collective agent, including the location of a conference, meeting, or voyage.
  • A number or other sequence designator of a conference, meeting, or voyage

Examples

has collective agent associated with manifestation
"Pearls Before Swine (Musical group)"
[Qualifiers: "(" + category of collective agent + ")".]
has collective agent associated with manifestation
"Workshop on Statistics and Computing in Disease Clustering (1992 : Port Jefferson, N.Y.)"
[Qualifiers: "(" + date of conference + ":" + place of conference + ")".]

A string encoding scheme should use any strings as qualifiers, if it is considered to be useful for users of the metadata.

Stipulations

Use the element super-type to record a less specific relationship.

Use an appropriate element sub-type to record a more specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.

Examples

has collective agent associated with manifestation
"United States. Works Progress Administration"
[Full example: National parks (2015; Ziga Media, LLC; flipchart). The value is an authorized access point of a collective agent who is an employer of the creators of collected works that are embodied in the manifestation.]
has collective agent associated with manifestation
"The Beatles"
[Full example: Revolver (2022; Apple Corps Limited; audio disc; 2 CD edition). The value is an authorized access point of a creator of an aggregating work and aggregated works that are embodied in the manifestation.]

Record a single occurrence of the element that is selected from one or more values that are associated with the manifestation. Use the following order of preference:

  1. Collective agent who is a creator of the manifestation
    1. Collective agent who is a publisher of the manifestation
    2. Collective agent who is a producer of the manifestation
    3. Collective agent who is a manufacturer of the manifestation
  2. Collective agent who is a distributor of the manifestation
  3. Collective agent who is associated with an expression or work that is embodied in the manifestation

Record additional occurrences of the element, if it is considered to be useful for users of the metadata.

Examples

has collective agent associated with manifestation
"Philip Glass Ensemble"
has collective agent associated with manifestation
"Lucinda Childs Dance Company"
[Full example: Einstein on the beach (2016; Opus Arte; videodisc; DVD). The values are authorized access points of creators of an expression that is embodied in the manifestation.]
