Related agent entities

Agent is an entity that is defined in the IFLA Library Reference Model as "An entity capable of deliberate actions, of being granted rights, and of being held accountable for its actions".

Collective Agent is an entity that is defined in the IFLA Library Reference Model as "A gathering or organisation of persons bearing a particular name and capable of acting as a unit".

Person is an entity that is defined in the IFLA Library Reference Model as "An individual human being".

Collective Agent and Person are entity sub-types of Agent. A person is an agent; a collective agent is an agent.

A collective agent comprises more than one person.
