Element reference

Definition
Relates a manifestation to a nomen that is an identifier that includes numbers or alphabetic characters to indicate a position in a sequence of units of a multiple unit manifestation or of iterations of a diachronic work.
Scope note
This element applies to a manifestation that embodies the expression of a diachronic work or an expression of an issue or iteration of a diachronic work.
Domain
Manifestation
Range
Nomen
Element sub-type
Element super-type
has identifier of manifestation

Additional information

A manifestation may be associated with more than one numeric designation in different languages, scripts, or sequences.

See also: has chronological designation

String encoding schemes

For a manifestation that embodies the expression of a diachronic work, a string encoding scheme should combine values for manifestations that embody expressions of issues or iterations of the diachronic work.

Numeric designation SES is a string encoding scheme suggested by ISBD for Manifestation for a value of this element.

Stipulations

Use an element super-type to record a less specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a value that is based on a designation in a statement of issue or iteration that appears in the manifestation or an external source of information.

For manifestation that embodies an expression of an issue or iteration of a diachronic work, record a value that is a numeric designation of only the issue or iteration.

Examples

has numeric designation
"Issue 357"
[Full example: Mojo 357 (2023; H Bauer Publishing Ltd.; volume + audio disc).]
has numeric designation
"Volume 3, issue 3"
[Full example: Quantum world (2016; Reed Business Information Ltd; volume).]

For a manifestation that embodies the expression of a diachronic work, record a value for the manifestation that embodies an expression of the first issue or iteration of the work in a numeric sequence, followed by punctuation to indicate an open sequence.

Examples

has numeric designation
"Issue 1-"
[Full example: Mojo (1993-; H Bauer Publishing Ltd.; volume + audio disc).]
has numeric designation
"Heft 1-"
[Full example: Journal für Facility Management (2009-; Technische Universität Wien‏; online resource).]

For a manifestation that embodies the expression of a diachronic work, and if there is a change in the sequence of numeric designations of issues or iterations, record a value that uses a string encoding scheme to combine:

  1. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the old numeric sequence
  2. a value for the manifestation that embodies an expression of the last issue or iteration of the work in the old numeric sequence
  3. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the new numeric sequence

Record a has note on nomen associated with manifestation element that gives information about the change, if it is considered to be useful for users of the metadata.

For a manifestation that embodies the expression of a diachronic work that is known or assumed to have terminated, record a value that uses a string encoding scheme to combine:

  1. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the numeric sequence
  2. a value for the manifestation that embodies an expression of the last issue or iteration of the work in the numeric sequence
