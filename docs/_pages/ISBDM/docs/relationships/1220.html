has work embodied in manifestation

Element reference

Definition
Relates a manifestation to a work that is carried by the manifestation.
Scope note
Domain
Manifestation
Range
Work
Element sub-type
Element super-type
has work associated with manifestation

Additional information

This element is a shortcut for:

has expression embodied in manifestation [Expression] realises Work

The shortcut does not identify the expression that is embodied in the manifestation. "Realizes"” is an Expression element that is in the inverse of LRM-R2 in the IFLA Library Reference Model.

See also: Preiminary assessment: Number of expressions

Stipulations

‼

Record at least one occurrence of the element, or record at least one occurrence of a has expression embodied in manifestation element.

Use the element super-type to record a less specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.

Examples

has work embodied in manifestation
"束漫,1971-.图书馆阅读障碍症人群服务研究"
[Full example: 图书馆阅读障碍症人群服务研究 (2019; 图家图书馆出版社; 册).]

For a manifestation that is an aggregate, and if more than one work is embodied in the manifestation, record one or more occurrences of the element, if it is considered to be useful for users of the metadata. Use the following order of preference to select the values:

  • The works that are named most prominently in the typography and layout of a manifestation statement of title and responsibility.
  • The work that is named first in a manifestation statement of title and responsibility.
  • The works that are named in a manifestation statement of title and responsibility in a language and script that is preferred by the cataloguing agency.
  • All works that are embodied in the manifestation.

Examples

has work embodied in manifestation
http://viaf.org/viaf/720154801932256310009
[Full example: Secret library (2016; Michael O’Mara Books Limited; volume; case binding). This is the more prominent work in an augmented expression aggregate.]

For a manifestation that is an aggregate, and if a collective title appears in any statement of title and responsibility or external source of information, record a value for the aggregating work that includes a collective title, if it is considered to be useful for users of the metadata.

Examples

has work embodied in manifestation
"Hamilton, Gill. Open licensing for cultural heritage"
[Full example: Open licensing for cultural heritage (2017; Facet Publishing; volume; perfect binding).]
has work embodied in manifestation
"كليلة ودمنة (بن المقفع، عبد الله)"
[Full example: كليلة ودمنة (1973؛ الشركة الوطنية للنشر والتوزيع؛ مجلد؛ غلاف مقوى). Authorized access point for an "anonymous classic".]

For a manifestation that embodies the expression of a diachronic work with a valid International Standard Serial Number (ISSN), for linked open data applications record a value that is the URI of the ISSN.

Examples

has work embodied in manifestation
https://portal.issn.org/resource/ISSN/2520-5404
[Full example: Journal für Facility Management (2009-; Technische Universität Wien‏; online resource).]

For a manifestation that embodies the expression of a diachronic work with a valid International Standard Serial Number (ISSN), for relational database applications record a value that is the ISSN.

Examples

has work embodied in manifestation
"ISSN 2813-2696"
[Full example: myScience Canada (2008-; Scimetrica; online resource).]
has work embodied in manifestation
"ISSN 2167-3241"
[Full example: RDA toolkit (2020-; RDA Copyright Holders; online resource).]

For a manifestation that embodies the expression of a diachronic work that does not have a valid International Standard Serial Number (ISSN), record a value that includes a common title and a category of carrier.

Examples

has work embodied in manifestation
"Springer handbooks of political science and international relations (online resource)"
[Full example: Springer handbooks of political science and international relations (2022-; Springer; online resource).]

For a manifestation that embodies an expression of an issue or iteration of a diachronic work, record a value that includes a common title, a designation and title of the sub-unit or iteration, and a category of carrier.

Examples

has work embodied in manifestation
"New scientist. The collection. Volume 3, issue 3 [volume]"
