has date of publication

Element reference

Definition
Relates a manifestation to a time-span during which the manifestation is published.
Scope note
Domain
Manifestation
Range
Time-span
Element sub-type
Element super-type
has date of creation of manifestation

Additional information

Record a date that appears in a statement of copyright as a date of publication, if it is considered to be useful for users of the metadata.

Stipulations

Use the element super-type to record a less specific relationship.

Apply the stipulations for the element super-type unless indicated otherwise.
