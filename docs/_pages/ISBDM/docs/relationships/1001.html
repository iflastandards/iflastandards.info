Additional information

This element supports the user tasks to find and identify the manifestation and to explore the context of the manifestation and its relationship to instances of the Work entity.

See also: has note on work associated with manifestation

Element values

The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.

String encoding schemes

[Guidance for base]

For a work that is anonymous, apply the guidance for the language or country of origin of the name that is available in Anonymous classics, if it is considered to be useful for users of the metadata.

A string encoding scheme should use any strings as qualifiers, if it is considered to be useful for users of the metadata.

Stipulations

Apply the General stipulations for relationship elements.

Use an appropriate element sub-type to record a more specific relationship.

Record a work that is associated with the manifestation. Do not record a work that is known to be falsely associated with the manifestation.

Record a separate occurrence of the element for each distinct value that is considered to be useful for users of the metadata.
