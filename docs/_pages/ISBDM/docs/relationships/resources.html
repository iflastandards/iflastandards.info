Related resource entities

Expression is an entity that is defined in the IFLA Library Reference Model as "A distinct combination of signs conveying intellectual or artistic content".

Item is an entity that is defined in the IFLA Library Reference Model as "An object or objects carrying signs intended to convey intellectual or artistic content".

Work is an entity that is defined in the IFLA Library Reference Model as "The intellectual or artistic content of a distinct creation".

A related manifestation is a manifestation that is distinct from the manifestation.
