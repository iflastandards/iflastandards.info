has number of sound channels

Element reference

Definition
Relates a manifestation to an extent of embodied content that is the number of distinct parallel streams that embody a single expression of audio content.
Scope note
This element applies to a manifestation that embodies performed music, sounds, or spoken word content.
Domain
Manifestation
Range
Element sub-type
Element super-type
has extent of embodied content

Additional information

Record the number of channels that embody the content. Do not include recording tracks that are not used.

Consider a ".1" channel for a subwoofer as a distinct channel.

See also: has designation of number of sound channels

See also: has number of recording tracks

Stipulations

Use the element super-type to record a less specific attribute.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a quantitative value that consists of a whole number and a term for "channel" in the language or script of choice of the cataloguing agency.
