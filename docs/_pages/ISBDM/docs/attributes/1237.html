has dimensions

Element reference

Definition
Relates a manifestation to an extent of unit that is a set of measurements of the size of the unit
Scope note
Dimensions include height, width, and depth.
Domain
Manifestation
Range
Element sub-type
Element super-type
has extent of unit

Additional information

This element does not apply to a manifestation that is an online resource.

An overall measurement includes a wrapper, binding, container, frame, mounting, or other component that is published or produced with the unit that does not embody content.

Element values

The value of this element is a set of two or three whole numbers that measure two or three dimensions and a controlled term or standard symbol for a unit of length.

Use the same unit of length for each dimension.

Round a fraction of the unit of length to the nearest whole number.

Record a has note on physical characteristic element that gives a more precise measurement, if it is considered to be useful for users of the metadata.

Record the number for each dimension in the stipulated order, and separate the numbers by the symbol "x" and spaces. Add the unit of length after the set of numbers.

Examples

has dimensions
"5 x 4 x 3 cm"
[Unit is 5 cm in height, 4 cm in width, and 3 cm in depth.]

A diameter represents the height and width dimensions with the same measurement.

A length of a cylinder represents the depth dimension.

Stipulations

Use the element super-type to record a less specific attribute.

Apply the stipulations for the element super-type unless indicated otherwise.

For a manifestation that is an aperture card, card, computer card, filmslip, filmstrip, microfiche, microfilm slip, microopaque, microscope slide, overhead transparency, sheet, slide, or stereograph card, record a value for the overall height and width of the unit.

Examples

has dimensions
"70 x 50 cm"
[Full example: Zadar does Banksy (2023; Narodni muzej Zadar; list). The sheet is 70 cm in height and 50 cm in width.]
has dimensions
"50 x 80 cm"
[Full example: Desserte rouge 1908 (1993; Editions du Centre Pompidou; sheet). The sheet is 50 cm in height and 80 cm in width.]

For a manifestation that is a sheet, and if the sheet is folded for publication or production, record a value for the overall height and width of the folded sheet or its wrapper or other container.

"28 x 15 cm"

For a manifestation that is an audio belt, audio cartridge, audiocassette, computer chip cartridge, computer tape cartridge, computer tape cassette, film cartridge, film cassette, filmstrip cartridge, microfiche cassette, microfilm cartridge, microfilm cassette, video cartridge, videocassette, or volume, record a value for the overall height and width of the unit. Include a value for the overall depth of the unit, if it is considered to be useful for users of the metadata.

For a manifestation that is an audio cylinder or computer disc cartridge, record a value for the overall diameter and length of the unit.

Examples

"62 x 110 mm"
[Unit is 62 mm in diameter and 110 mm in length.]
