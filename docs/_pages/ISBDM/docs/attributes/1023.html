has extent of manifestation

Element reference

Definition
Relates a manifestation to a physical characteristic that is a quantification of the physical characteristics of the manifestation.
Scope note
Domain
Manifestation
Range
Element sub-type
has extent of aggregated content has extent of embodied content has extent of unit has extent of unitary structure
Element super-type
has physical characteristic

Additional information

This element supports the user tasks to identify and select the manifestation.

The values of this element may be indexed for controlled keywords.

The values of this element may be parsed to extract carrier and content metrics.

See also: has note on manifestation statement of extent

Element values

The value of this element is a quantitative value that measures a single characteristic, or a combination of quantitative and qualitative values that measure multiple characteristics.

Apply a string encoding scheme to construct a combination of values.

String encoding schemes

A string encoding scheme should combine values of the element sub-types in the following order:

  1. has extent of unitary structure
  2. has extent of unit
  3. has extent of aggregated content
  4. has extent of embodied content

Examples

has extent of manifestation
"3 volumes (124 leaves; 150 photographs; 200 pages)"
has extent of manifestation
"3 volumes (150 photographs; 200 pages)"
has extent of manifestation
"3 volumes (200 pages)"

For a manifestation that has no sub-units, use the string that is the number "1" followed by a value for the has category of carrier element as a substitute for a value of has extent of unitary structure.

Examples

has extent of manifestation
"1 volume (123 pages)"

For a manifestation that is a super-unit, a string encoding scheme may combine values of element sub-types for sub-units of the manifestation.

Examples

elementLabel
"2 audio discs (12 in; 33 rpm; 12 studio music performances; 42 mins; stereo + 12 in; 33 rpm; 4 live music performances; 30 mins; mono)"

For a manifestation that embodies only one expression, use the string that is the number "1" followed by a value for the has category of embodied content element or a controlled term for a finer level of granularity of the category as a substitute for a value of has extent of aggregated content.

Examples

has extent of manifestation
"1 sheet (1 page, 1 drawing)"

Stipulations

Apply the General stipulations for attribute elements.

Use an appropriate element sub-type to record a more specific attribute.

Record a quantitative value that consists of a whole number and a controlled term or standard symbol for a unit of a measurement or other quantifiable characteristic of the manifestation.

Examples

has extent of manifestation
"1 drawing"

Record a value that is a combination of quantitative and qualitative values that measure multiple characteristics and that is constructed by applying a string encoding scheme.

Examples

has extent of manifestation
"2 CDs (15 studio music performances; 72 mins; stereo + 14 studio music performances; 70 mins; stereo) + 1 booklet (12 pages; 2 photographs)"
