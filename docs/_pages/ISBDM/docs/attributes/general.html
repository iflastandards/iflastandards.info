General stipulations for attribute elements

Record an attribute if it is considered to be useful for users of the metadata, unless indicated otherwise.

Record an attribute that is named or otherwise identified in a statement that appears in the manifestation.

Record an attribute that is associated with the manifestation in external sources of information.

Record an attribute of the manifestation. Do not record an attribute that is known to be falsely associated with the manifestation.

Record an attribute that is real and is known to exist or have existed. Do not record an attribute that is known to be fictitious.

Record a has note on manifestation element that gives information that is not recorded in an attribute element, if it is considered to be useful for users of the metadata.

If more than one value is associated with the same attribute of the manifestation, record a separate occurrence of the element for each distinct value that is considered to be useful for users of the metadata. Use the following order of preference:

  1. The predominance of values that are given in external sources of information.
  2. The prominence of values that apply to the manifestation.
  3. The order of values that appear in a statement in the manifestation.

For a manifestation that embodies the expression of a diachronic work, and if there is a change in the information that is given in statements that describe the manifestation and appear in manifestations that embody expressions of issues or iterations of the work, record an additional occurrence of the element for the attribute that has changed, if it is considered to be useful for users of the metadata.

For a manifestation that embodies the expression of a diachronic work, record a has note on category of manifestation element that gives information about a change in an attribute that is common to the manifestations that embody expressions of issues or iterations of the work, including identification of the issue or iteration in which the change first appears, if it is considered to be useful for users of the metadata.

For a manifestation that embodies the expression of a diachronic work, record a has note on category of manifestation element that gives general information about multiple changes over time in an attribute that is common to the manifestations that embody expressions of issues or iterations of the work, including identification of the issues or iterations in which the changes occur, if it is considered to be useful for users of the metadata.

Element values

The value of an attribute element is a term or phrase that refers to a concept that is associated with a characteristic of the manifestation. The value may be qualitative or quantitative.

  • A qualitative value is a controlled term that appears in a value vocabulary.
  • A quantitative value is a measurement that consists of a number and a qualitative term for the unit of measurement.

Record a qualitative value of an attribute element as a literal string or as an Internationalized Resource Identifier (IRI) that is used in semantic linked data.

Record only one kind of value for a single instance of an attribute.

Record different kinds of values for different occurrences of elements, if it is considered to be useful for users of the metadata.

For a linked open data application, use the following order of preference to select the value:

  1. an IRI for the concept that appears in a value vocabulary
  2. a string that is an identifier of the concept that is a stringified IRI for the concept, derived by enclosing the IRI in string delimiters such as double quotes ("IRI")
  3. a string that is an identifier or controlled term for the concept that appears in a value vocabulary
  4. a string that is an uncontrolled term for the concept

For a relational database application, use the following order of preference to select the value:

  1. a string that is an identifier or controlled term for the concept that appears in a value vocabulary
  2. a string that is an uncontrolled term for the concept

Record a quantitative value of an attribute element as a literal string that is constructed from a number and a controlled term that appears in a value vocabulary.

Sources of information

Use The manifestation or External sources as a source of information for an attribute element.
