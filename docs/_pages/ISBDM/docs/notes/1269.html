has note on manifestation associated with manifestation

Element reference

Definition
Relates a manifestation to a note on manifestation that provides information about a manifestation that is related to the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type
Element super-type
has note on entity associated with manifestation

Additional information

See also: has manifestation associated with manifestation

Stipulations

Use the element super-type to record a less specific note.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a value that gives information about other manifestations that embody the same content in a different category of carrier.

Examples

has note on manifestation associated with manifestation
"Also published online and on microfilm."
[String value]
has note on manifestation associated with manifestation
"Also issued as glass slides."
[String value]

Record a has alternate manifestation element that relates the manifestation to another manifestation that is published or produced at the same time and that embodies the same content.

Record a value that gives information about the bibliographic history of the manifestation, including previous manifestations that are related to the manifestation.

Examples

has note on manifestation associated with manifestation
"Originally published: London : J. Tonson, 1747."
has note on manifestation associated with manifestation
"Previously published as: The players of Null-A. London : Dodson, 1970; originally published as: The pawns of Null-A. New York : Street and Smith, 1948."

For a manifestation that reproduces another manifestation, record a value that gives information about the manifestation that is reproduced.

Examples

has note on manifestation associated with manifestation
"Facsimile reproduction. Originally published: London : R. Walton, 1654."
has note on manifestation associated with manifestation
"Reproduction of a drawing created in 1522."

For a manifestation that is published or produced in more than one unit, record a value that gives information about one or more sub-units or super-units.

Examples

has note on manifestation associated with manifestation
"On two discs with an accompanying booklet."
has note on manifestation associated with manifestation
"Part of a set of two discs with an accompanying booklet."

For a manifestation that embodies an expression of an issue of a diachronic work, record a value that gives information about the manifestation that embodies the expression of the diachronic work.

Examples

has note on manifestation associated with manifestation
"Issue of New scientist. The collection (print)."
has note on manifestation associated with manifestation
"First update for the 1973 volume."

For a manifestation that embodies the expression of a diachronic work, record a value that gives information about the manifestations that embody the expressions of issues of the diachronic work.

Examples

has note on manifestation associated with manifestation
"Volume 3, issue 3 has title The quantum world."
