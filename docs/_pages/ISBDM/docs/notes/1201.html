has note on manifestation statement of edition

Element reference

Definition
Relates a manifestation to a note on manifestation statement that provides information about a statement of edition that is transcribed from the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type
Element super-type
has note on manifestation statement

Additional information

See also: has manifestation statement of edition

Stipulations

Use the element super-type to record a less specific note.

Apply the stipulations for the element super-type unless indicated otherwise.

Examples

has note on manifestation statement of edition
"Version en français."
[Full example: Algerie a travers les ages (2000; Cerist; disque informatique; version française).]

Record a value that gives information about a statement of edition that appears in the manifestation but does not apply to the manifestation as a whole.
