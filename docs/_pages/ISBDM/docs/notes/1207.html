has note on manifestation

Element reference

Definition
Relates a manifestation to information about the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type
has note on category of manifestation has note on entity associated with manifestation has note on manifestation statement has note on physical characteristic
Element super-type

Additional information

This element supports the user task to identify the manifestation.

Element values

The values of this element may be indexed for uncontrolled keywords to support the user task to find the manifestation.

For a manifestation that embodies the expression of a diachronic work, include a value of any of the following elements for the identification in a note of the manifestation that embodies an expression of an issue or iteration of the work:

  • has chronological designation
  • has date of publication
  • has numeric designation

Use a has note on item associated with manifestation element to record information about a modification of an item that exemplifies the manifestation after its publication or production.

Stipulations

Apply the General stipulations for note elements.

Use an appropriate element sub-type to record a more specific note.

Examples

has note on manifestation statement
"Statements are taken from the journal website and the cover page of the first issue, vol. 1, no. 1 April 2022."
[Full example: Journal of construction automation and robotics (2022-; Korean Society of Automation and Robotics in Construction; online resource). The note is about statements that appear in the manifestation.]
has note on category of manifestation
"Същото съдържание е въплътено в печатен том и DVD."
[Full example: Съкровищница (2018; Народна библиотека “Св. св. Кирил и Методий”; том + компютърен диск). The note is about the categorization of the manifestation.]
has note on physical characteristic
"The text is laid out in 2 columns on each page."
[Full example: Black Lizard big book of locked-room mysteries (2014; Vintage Books; volume; perfect binding). The note is about the layout of the content that is embodied in the manifestation.]

If a statement that appears normally in a manifestation is absent, record a value that gives information that appears in an external source of information.

Examples

has note on manifestation
"Bez podataka o izdavanju; nakladnik je Narodni muzej Zadar (NMZ). Izložba se održava u Zadru 2023."
[Full example: Zadar does Banksy (2023; Narodni muzej Zadar; list).]
has note on manifestation
"Published by Biblioteka Kombëtare e Shqipërisë, Tiranë, 2011."
has note on manifestation
"Manufactured by Argjendari Pirro."
[Full example: 90 vjet Bibliotekë Kombëtare (2011; Biblioteka Kombëtare e Shqipërisë; object).]

For a manifestation that is published, record a value that gives information about intended variations in the items that exemplify the manifestation.

Examples

has note on manifestation
"Each copy is numbered and signed."
has note on manifestation
"First 25 copies are bound in leather."

For a manifestation that embodies the expression of a diachronic work, record a value that gives information about the intended or operational termination of the diachronic work.

Examples

has note on manifestation
"Latest issue available from publisher’s website is volume 6, issue 5, December 2019, checked on 27 April 2022."
[Full example: New scientist. The collection (2014-; Reed Business Information Ltd; volume).]
has note on manifestation
"Izdavanje je prestalo 1998."
[Full example: Biblioteka Vjeverica (1957-1998, Mladost; svezak).]
