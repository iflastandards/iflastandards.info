has note on manifestation statement

Element reference

Definition
Relates a manifestation to a note on manifestation that provides information about a manifestation statement that is transcribed from the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type

Element super-type
has note on manifestation

Additional information

See also: has manifestation statement

Stipulations

Use the element super-type to record a less specific note.

Use an appropriate element sub-type to record a more specific note.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a value that is a transliteration in the script of choice of the cataloguing agency of a statement that appears in the manifestation in a different script, including identification of the script that appears in the manifestation.

Record a value that is a translation in the language of choice of the cataloguing agency of a statement that appears in the manifestation in a different language, including identification of the language that appears in the manifestation.

Record a value that gives information about a statement that is duplicated in the manifestation in a different language or script, including identification of the languages or scripts that appear in the manifestation.

Record a value that is derived from a statement that appears in the manifestation but is not recorded in a has manifestation statement element or element sub-type.

Record a value that gives information about the absence of a statement that normally appears in a manifestation.

Examples

has note on manifestation statement
"No statements appear in the manifestation."
[Full example: Pogled na Akademijin vrt (195-; Mujadžić, Omer, 1903-1991; sheet). The manifestation embodies a painting that does not include a signature, title, caption, or other text.]

Record a value that gives the source of information of a manifestation statement.

Examples

has note on manifestation statement
"Statements taken from volume."
[Full example: Mojo 357 (2023; H Bauer Publishing Ltd.; volume + audio disc). The manifestation has volume and audio disc sub-units.]
has note on manifestation statement
"Statements taken from the first issue."
[Full example: Springer handbooks of political science and international relations (2022-; Springer; online resource).]
has note on manifestation statement
"Statements taken from the reverse side of the card."
[Full example: Truman Capote, ecrivain, USA, 1947 (2008; Nouvelles Images S.A.; card).]

Record a value that gives an explanation or description of a character, ideograph, or punctuation mark that is inverted or reversed and that is transcribed in its correct orientation in the value of a recorded manifestation statement.

Examples

has note on manifestation statement
"The first “B” in “ABBA” is reversed."

Record a value that gives the scope of omissions in a recorded manifestation statement, such as the number of omitted words, names or titles.

Record a value that gives the full forms of abbreviations of words that appear in a recorded manifestation statement.

Examples

has note on manifestation statement
"Boetij viri celeberrimi de [con]solatio[n]e phylosophie liber : cu[m] optimo [com]me[n]to beati Thome."

Record a value that gives an explanation or description of the symbol, sign, rebus, or image that is replaced in the value of a recorded manifestation statement.

Examples

has note on manifestation statement
"Copyright symbol replaced by “c”."
has note on manifestation statement
"Copyright symbol replaced by “copyright”."
has note on manifestation statement
"Phonogram symbol replaced by “&;."
has note on manifestation statement
"“E.B.C.” is represented as a sequence of musical notes."

Record a value that gives an explanation or description of a symbol or sign that is omitted in the value of a recorded manifestation statement.

If the information that is transcribed in a manifestation statement is known to be incorrect, record a value that gives a correction with an explanation.

Examples

has note on manifestation statement
"Looser is a misspelling of loser."
has note on manifestation statement
"Hnmble is a misprint of humble."

If the information that is transcribed in a manifestation statement is known to be fictitious, record a value that gives an explanation.

For a manifestation that is an older printed sheet or volume or a manifestation that is produced, record a value that indicates the position of a letter or word that is intentionally left blank in a statement that appears in the manifestation.

Examples

has note on manifestation statement
"Blank spaces not filled in for day and final year of date: Gifwen Stockholms Rådhus den [] Junii 176[]."
has note on manifestation statement
"Blank spaces filled in for day and final year of date."

For a manifestation that is an older printed sheet or volume, record a value that gives information about variations in the state of the manifestation.

Examples

has note on manifestation statement
"Variant state has misprint in colophon: Stocholm."

For a manifestation that is an older printed sheet or volume, record a value that indicates a conjectural expansion of an abbreviation that appears in a recorded manifestation statement.

For a manifestation that embodies the expression of a diachronic work, record a value that gives information about a change in statements that describe the manifestation and appear in manifestations that embody expressions of issues or iterations of the work, including identification of the issue or iteration in which the change first appears.

For a manifestation that embodies the expression of a diachronic work, record a value that gives general information about multiple changes over time in statements that describe the manifestation and appear in manifestations of expressions of issues or iterations of the work, including identification of the issues or iterations in which the changes occur.
