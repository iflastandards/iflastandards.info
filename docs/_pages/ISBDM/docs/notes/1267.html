has note on expression associated with manifestation

Element reference

Definition
Relates a manifestation to a note on entity associated with manifestation that provides information about an expression that is related to the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type
Element super-type
has note on entity associated with manifestation

Additional information

Record a value of a broad kind of expression that is embodied in a manifestation as a value of has category of embodied content.

For a manifestation that is an aggregate, record a quantitative value for a kind of expression that is embodied in the manifestation as a value of has extent of aggregated content.

See also: has category of embodied content

See also: has expression associated with manifestation

See also: has extent of aggregated content

Stipulations

Use the element super-type to record a less specific note.

Apply the stipulations for the element super-type unless indicated otherwise.

For a manifestation that embodies an expression that is a translation of another expression, record a value that gives information about the expression that is translated.

Examples

has note on expression associated with manifestation

[Full example: In a flight of starlings (2023; Allen Lane; volume).]

For manifestation that is an aggregate, record a value that gives information about one or more of the expressions that are embodied in the manifestation.
