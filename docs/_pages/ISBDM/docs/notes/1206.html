has note on manifestation statement of title and responsibility

Element reference

Definition
Relates a manifestation to a note on manifestation statement that provides information about a statement of title and responsibility that is transcribed from the manifestation.
Scope note
Domain
Manifestation
Range
Literal
Element sub-type
Element super-type
has note on manifestation statement

Additional information

See also: has manifestation statement of title and responsibility

Stipulations

Use the element super-type to record a less specific note.

Apply the stipulations for the element super-type unless indicated otherwise.
